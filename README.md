OOP (Object-Oriented Programming) nima?

OOP — bu dasturlash paradigmasi, ya’ni dastur tuzish usuli bo‘lib, unda hamma narsa obyektlar orqali ifodalanadi. Har bir obyekt real hayotdagi biror narsaning modeli bo‘lib, u xususiyatlar (fields) va xatti-harakatlar (methods) ga ega bo‘ladi.

Masalan:
Talaba — obyekt

xususiyatlari: ism, yosh, baho

xatti-harakatlari: o‘qish, imtihon topshirish

OOP ning asosiy maqsadi

OOP ning asosiy maqsadi:

kodni tushunarli

qayta ishlatiladigan

kengaytirish oson

va xavfsiz

qilishdir.

OOP ning asosiy tushunchalari
Class (Sinf)

Class — bu obyekt uchun shablon (reja).
U obyekt qanday xususiyatlarga va methodlarga ega bo‘lishini belgilaydi.

Masalan: Car classi — mashina uchun reja.

Object (Obyekt)

Object — bu class asosida yaratilgan real nusxa.

Masalan:
Car c1 = new Car();
Bu — Car classidan yaratilgan obyekt.

OOP ning 4 ta asosiy prinsipi
1. Encapsulation (Inkapsulyatsiya)

Encapsulation — ma’lumotlarni yashirish va ularga faqat methodlar orqali murojaat qilish.

Bu:

noto‘g‘ri foydalanishdan himoya qiladi

kodni xavfsiz qiladi

Masalan, balance o‘zgaruvchisi private bo‘lib, faqat deposit() yoki withdraw() orqali o‘zgartiriladi.

2. Inheritance (Meros olish)

Inheritance — bir class boshqa classning xususiyat va methodlarini meros qilib olishi.

Bu:

kodni qisqartiradi

umumiy funksiyalarni qayta yozmaslikka yordam beradi

Masalan:
Student classi → Person classidan meros oladi.

3. Polymorphism (Ko‘p shakllilik)

Polymorphism — bitta method nomi, lekin har xil ishlash usuli.

Bu holat:

methodni override qilish orqali amalga oshadi

dasturni moslashuvchan qiladi

Masalan:
draw() methodi — Circle da boshqacha, Rectangle da boshqacha ishlaydi.

4. Abstraction (Abstraksiya)

Abstraction — foydalanuvchiga faqat kerakli qismni ko‘rsatish, ichki murakkablikni yashirish.

Bu:

abstract class

yoki interface

orqali amalga oshiriladi.

Masalan:
Mashina haydashda biz dvigatel qanday ishlashini emas, faqat rul va pedalni bilamiz.

Xulosa

OOP — zamonaviy dasturlashning asosi bo‘lib, ayniqsa Java, C++, C#, Python kabi tillarda keng qo‘llaniladi. OOP yordamida real hayotni dastur orqali mantiqan va tartibli tarzda modellashtirish mumkin.
